# 1. Linux 重定向

## 标准输出

### `>` 覆盖

把正确输出写入文件，会覆盖原内容。

```
echo hello > test.txt
```

结果：

```
test.txt 原内容被清空
```

------

### `>>` 追加

把正确输出追加到文件末尾。

```
echo hello >> test.txt
```

结果：

```
原内容保留，新增内容追加
```

------

## 标准错误输出

### `2>` 覆盖错误信息

只接收错误输出。

```
ls aaa 2> error.txt
```

------

### `2>>` 追加错误信息

```
ls aaa 2>> error.txt
```

------

总结：

| 符号  | 作用         |
| ----- | ------------ |
| `>`   | 正确输出覆盖 |
| `>>`  | 正确输出追加 |
| `2>`  | 错误输出覆盖 |
| `2>>` | 错误输出追加 |

# 2. 查看命令执行状态

```
echo $?
```

查看上一条命令执行结果。

返回：

```
0
```

表示成功。

非 0：

表示失败。

例：

```
ls
echo $?
```

# 3. head 查看文件头部

默认查看前 10 行：

```
head 文件
```

指定行数：

```
head -n 5 文件
```

查看前 5 行。

------

# 4. tail 查看文件尾部

默认查看最后 10 行：

```
tail 文件
```

指定行：

```
tail -n 20 文件
```

查看最后20行。

------

## tail 实时监控

### `-f`

实时查看文件新增内容：

```
tail -f log.txt
```

常用于：

- 日志监控
- 服务排查

### `-F`

监控不存在或重新生成的文件：

```
tail -F log.txt
```

比 `-f` 更稳定。

------

# 5. head + tail 组合查看指定行

查看第5行到第10行：

方法：

```
head -n 10 file | tail -n 6
```

或者：

```
tail -n +5 file | head -n 6
```

```

## 反向排序

```

# 三、Windows 双网络路由问题

场景：

电脑同时连接：

- 教室局域网
- 手机热点

路由：

```
教室局域网：

192.168.18.x

用于：
VNC
内网网页


手机热点：

10.114.138.x

用于：
互联网下载
```

------

查看路由：

```
route print
```

发现：

```
0.0.0.0
```

存在多个默认路由。

例如：

```
0.0.0.0  192.168.18.1      Metric 35

0.0.0.0  10.114.138.241    Metric 35
```

Windows 根据：

> 接口跃点数（Metric）

选择出口。

数字越小：

优先级越高。

------

修改手机热点优先：

设置：

```
10.114.138.x

接口跃点数：
10
```

让：

```
互联网
 ↓
手机热点
```

------

测试：

IPv4：

```
tracert -4 www.baidu.com
```

显示：

```
10.114.138.241
```

说明：

走手机热点。

------

## 局域网访问恢复

问题：

手机热点成为默认出口后：

访问：

```
192.168.11.253
```

失败。

原因：

没有对应路由。

添加：

```
route add 192.168.11.0 mask 255.255.255.0 192.168.18.1
```

作用：

```
访问192.168.11.x

↓

走192.168.18.1

↓

教室局域网
```

# 四、SELinux 配置

配置文件：

```
/etc/selinux/config
```

查看：

```
cat /etc/selinux/config
```

当前：

```
SELINUX=enforcing
SELINUXTYPE=targeted
```

------

## 修改关闭 SELinux

命令：

```
sed -i '7c SELINUX=disabled' /etc/selinux/config
```

解释：

```
7
↓
第7行

c
↓
change 替换

SELINUX=disabled
↓
替换内容
```

效果：

把：

```
SELINUX=enforcing
```

修改：

```
SELINUX=disabled
```

更安全写法：

```
sed -i 's/^SELINUX=.*/SELINUX=disabled/' /etc/selinux/config
```
