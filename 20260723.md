# Linux 常用命令与软件包管理笔记

------

# 一、diff 文件比较

## 1. diff 基本格式

```
diff 文件1 文件2
```

例如：

```
diff 1.txt 2.txt
```

输出：

```
1,3c1,4
< 1
< 2
< 3
---
> a
> b
> c
>
```

------

## 2. diff 输出说明

格式：

```
旧文件行号 操作 新文件行号
```

例如：

```
1,3c1,4
```

含义：

- `1,3` ：旧文件第 1～3 行
- `c` ：change，修改
- `1,4` ：新文件第 1～4 行

即：

> 旧文件第 1～3 行内容，需要替换成新文件第 1～4 行内容。

------

## 3. diff 常见符号

| 符号 | 含义        |
| ---- | ----------- |
| a    | add 增加    |
| d    | delete 删除 |
| c    | change 修改 |
| <    | 旧文件内容  |
| >    | 新文件内容  |

------

# 二、文件统计命令

## 1. wc -l 统计行数

格式：

```
wc -l 文件名
```

例：

```
wc -l 1.txt
```

输出：

```
3 1.txt
```

表示：

文件共有 3 行。

------

## 2. du 查看磁盘占用

### 查看文件/目录大小

```
du -h
```

参数：

```
-h
human-readable
```

显示易读单位：

```
K
M
G
```

例如：

```
16K ./.gnupg
76K .
```

------

## 3. 只显示目录总大小

```
du -sh
```

参数：

| 参数 | 作用                |
| ---- | ------------------- |
| -s   | summary，只显示汇总 |
| -h   | 人类可读格式        |

例如：

```
du -sh /root
```

结果：

```
76K /root
```

# 三、YUM 软件包管理

## 1. yum clean all

清理 yum 缓存：

```
yum clean all
```

作用：

- 清理下载的软件包缓存
- 清理仓库缓存信息
- 释放磁盘空间

------

# 四、yum 常用参数

## 1. 自动确认

```
-y
```

作用：

自动回答：

```
Is this ok [y/N]
```

中的：

```
y
```

例：

```
yum -y install wget
```

------

## 2. 只下载不安装

参数：

```
--downloadonly
```

例：

```
yum -y install --downloadonly wget
```

------

## 3. 指定下载目录

参数：

```
--downloaddir=
```

例：

```
yum -y install \
--downloadonly \
--downloaddir=/tmp \
wget
```

作用：

只下载 rpm 包到 `/tmp`

不安装。

# 五、查看网络端口

## netstat

查看端口：

```
netstat -tnulp
```

参数：

| 参数 | 含义         |
| ---- | ------------ |
| -t   | TCP          |
| -n   | 显示数字端口 |
| -u   | UDP          |
| -l   | 监听状态     |
| -p   | 显示进程     |

示例：

```
tcp  0 0 0.0.0.0:8080 LISTEN 1234/java
```

表示：

Java 程序监听 8080 端口。

# 六、yum 搜索软件包

问题：

安装命令不存在：

```
rz: command not found
```

原因：

可能：

1. 软件包名称错误
2. 命令属于其他安装包

解决：

搜索软件属于哪个包：

```
yum search sz
```

例如：

找到：

```
lrzsz
```

安装：

```
yum install -y lrzsz
```

安装后：

```
rz
sz
```

即可使用。

# 七、RPM 软件包管理

## 1. RPM 特点

类似 Windows：

```
.exe
```

安装包。

特点：

1. 先下载 `.rpm`
2. 再安装
3. 不需要网络
4. 不自动解决依赖

------

# 2. rpm 安装

语法：

```
rpm -ivh xxx.rpm
```

参数：

| 参数 | 作用         |
| ---- | ------------ |
| -i   | install 安装 |
| -v   | 显示安装过程 |
| -h   | 显示进度条   |

例：

```
rpm -ivh wget.rpm
```

------

# 3. 查询 RPM

## 查看是否安装

```
rpm -q 软件名
```

例：

```
rpm -q wget
```

------

## 查看所有已安装软件

```
rpm -qa
```

例如：

```
rpm -qa | grep nginx
```

------

## 查看安装文件位置

```
rpm -ql 软件名
```

例：

```
rpm -ql wget
```

查看：

```
/usr/bin/wget
```

------

## 查看配置文件

```
rpm -qc 软件名
```

例如：

```
rpm -qc ssh
```

查看配置：

```
/etc/ssh/sshd_config
```

------

# 4. RPM 卸载

方法1：

```
rpm -e 软件名
```

例如：

```
rpm -e wget
```

方法2：

使用 yum：

```
yum -y remove wget
```

------

# 八、RPM 安装总结

安装：

```
rpm -ivh xxx.rpm
```

查询：

```
rpm -q 软件名
```

所有软件：

```
rpm -qa
```

```
rpm -ql 软件名
```

查看配置：

```
rpm -qc 软件名
```

卸载：

```
rpm -e 软件名
```

# 九、Ubuntu 软件包管理

## RPM 和 DEB 区别

| 系统          | 软件包 |
| ------------- | ------ |
| CentOS/RHEL   | .rpm   |
| Ubuntu/Debian | .deb   |

Ubuntu：

❌ 不能安装：

```
xxx.rpm
```

使用：

```
xxx.deb
```

------

# 十、dpkg 管理 deb 包

## 安装

```
dpkg -i xxx.deb
```

例如：

```
dpkg -i nginx.deb
```

------

## 卸载

```
dpkg -r 软件名
```

或者：

```
apt -y remove 软件名
```
